## Porter Color Palette:

![porter-palette](styleguide/porter-palette.png)

### Color values: 

Primary (brand elements):

```
$gray:   #3C2E53; // ideal for text
$gold:    #DFB960;
```

Secondary (backgrounds, shading, accents):

```
$green:  #DDF5F2; // ideal for bg
$pink:   #CFBFD0;
```

Cat colors:

```
$red:     #AB2449;
$orange:  #F1CE91;
```
